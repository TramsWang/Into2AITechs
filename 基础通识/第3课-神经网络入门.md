# 第3课：神经网络入门

## 内容引入

**3 分钟互动：一眼认图为什么不简单？**

教师展示一张较模糊的动物图片，先不公布答案，让学生快速说出判断依据：耳朵、眼睛、尾巴、体型、毛色等。不同学生可能关注不同特征。

引导学生思考：人类识别图片时会综合很多线索，机器也需要一种能综合复杂特征的模型。

## 知识点讲授

### 1. 为什么需要神经网络

简单问题可以用简单规则解决，例如分数是否及格、年龄是否达到标准、温度是否超过某个数值。但复杂问题往往很难靠几条规则解决：

- 识别一张图片里是不是猫；
- 判断一句话表达的是开心还是难过；
- 翻译一段文字；
- 写一篇结构完整的文章；
- 从语音中识别出文字。

这些任务包含大量细节和变化，需要模型能够组合很多线索。

### 2. 神经网络像什么

神经网络可以类比为一个“班级投票系统”：

- 每个同学根据自己看到的一部分信息给出判断；
- 有些同学观察到的线索更关键，意见权重更大；
- 最后把许多判断综合起来，得到结果。

神经网络中的每个节点都做一点简单计算，许多节点连接在一起，就能完成复杂判断。

### 3. 输入层、隐藏层、输出层

可以把神经网络分成三类部分：

```text
输入层：接收信息
隐藏层：加工信息
输出层：给出结果
```

例如识别猫狗图片：

```text
图片像素 → 神经网络处理 → 输出“猫”或“狗”
```

输入层接收图片数字信息，隐藏层逐步提取边缘、形状、局部特征和整体特征，输出层给出最终判断。

### 4. 参数和训练

参数可以理解为“每条信息的重要程度”。如果某些特征对判断更关键，模型就应该给它们更高权重。

训练过程就是不断调整这些参数。模型先做预测，再和正确答案比较，如果错了，就调整参数；反复多次之后，模型的答案会逐渐接近正确答案。

### 5. 神经网络不是人脑复制品

“神经网络”这个名字借鉴了人脑神经元的启发，但课堂中不需要把它理解成人脑。更准确地说，它是一种由很多简单计算单元连接起来、可以通过训练调整参数的数学模型。

## 互动强化

**20 分钟活动：全班模拟神经网络**

### 环境一：每个学生都可以使用自己的电子设备

1. 教师在屏幕上展示 4 张动物图片，每张图片短暂显示。
2. 学生用设备填写自己负责观察的特征，例如耳朵形状、尾巴长度、体型、毛色。
3. 教师把全班结果汇总成表格，展示不同特征对判断的帮助程度。
4. 小组讨论：哪些特征更重要？哪些特征会误导判断？
5. 教师设置“权重”：关键特征投票算 2 分，普通特征算 1 分，再重新判断。
6. 总结：神经网络通过许多节点和参数，把局部信息组合成整体判断。

### 环境二：只有老师可以使用电子设备

1. 教师投影一张模糊动物图，把全班分为“耳朵组”“眼睛组”“尾巴组”“体型组”。
2. 每组只关注自己的特征并给出判断。
3. 教师在黑板记录各组判断，并给不同特征临时设置权重。
4. 全班根据加权结果得到最终答案。
5. 教师换一张图片重复一次，让学生感受权重调整会影响输出。

### 环境三：不能使用电子设备

1. 教师准备纸质描述卡，例如“耳朵尖”“尾巴短”“体型小”“会汪汪叫”等。
2. 每组抽取若干特征卡，判断对应动物可能是什么。
3. 教师给每张特征卡设置分值，例如“会汪汪叫”对狗的判断权重更高。
4. 学生根据分值重新计算判断结果。
5. 教师总结：节点像一个个小判断器，参数像重要程度，训练就是调整重要程度。

## 课后提升

**20 分钟小作业：画出一个迷你神经网络**

选择一个判断任务，例如“判断一部电影是否适合周末观看”或“判断一张动物图片是猫还是狗”。画出：

```text
输入特征 → 隐藏层处理 → 输出结果
```

至少写出 4 个输入特征，并说明你认为哪个特征最重要。能力较强的同学可以查阅“权重”和“偏置”的简单含义，用一句话解释它们在模型中的作用。

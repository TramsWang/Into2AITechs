# 第4课：文字如何变成机器能懂的数字？

## 前一节课内容回顾

**3 分钟回顾：从“神经网络”到“语言输入”**

上一节课我们学习了神经网络的基本结构：输入层接收信息，隐藏层加工信息，输出层给出结果；参数可以理解为不同线索的重要程度，训练就是让模型在预测和纠错中不断调整这些参数。同时要强调，神经网络不是人脑复制品，而是一种由许多简单计算单元连接起来的数学模型。

作业点评时，可表扬结构清楚、特征选择合理的迷你神经网络图。例如优秀作业会把“判断一部电影是否适合周末观看”画成“题材、时长、评分、同伴兴趣 → 隐藏层综合判断 → 适合/不适合”，并说明“同伴兴趣”或“时长”为什么权重更高。这类作业优秀之处在于不只是画了层次，还能解释不同输入特征怎样影响输出结果。

承接上一课的问题：神经网络需要接收输入才能计算。图片可以变成像素数字，声音可以变成波形数字，那么人类文字怎样进入模型？这节课讨论文字如何变成机器能懂的数字。

## 内容引入

**3 分钟互动：电脑真的懂“空落落”吗？**

教师在黑板写下主文本中的几个词：“进步”“笑了很久”“担心”“空落落”。让学生快速说出这些词可能让人联想到什么：努力得到肯定、表面开心、考试压力、说不清的失落感等。

随后提问：“电脑看到‘空落落’这三个字时，真的知道这是一种心里不踏实、不满足的感觉，还是只看到了某种编码？”

引导学生进入核心问题：计算机不能直接理解人类文字，必须先把文字转化成数字。

## 知识点讲授

### 1. 计算机只处理数字

人类看到“空落落”会联想到一种失落、不踏实的心理状态，看到“进步”会联想到努力和结果变好，但计算机底层处理的是数字。文字、图片、声音进入计算机后，都需要变成数字形式才能被存储和计算。

所以，AI 处理语言的第一步不是“理解”，而是把文字转换成可计算的表示。

### 2. 最简单的文字编码

一种直接做法是给每个字符一个编码。比如把“我心里空落落的”拆成一个个字符后，在 UTF-8 编码中，这几个汉字会被表示成不同的字节。UTF-8 常用十六进制表示，换成十进制后也是一组普通数字：

| 字符 | UTF-8 十六进制字节 | 对应的十进制字节 |
| ---- | ------------------ | ---------------- |
| 我   | E6 88 91           | 230 136 145      |
| 心   | E5 BF 83           | 229 191 131      |
| 里   | E9 87 8C           | 233 135 140      |
| 空   | E7 A9 BA           | 231 169 186      |
| 落   | E8 90 BD           | 232 144 189      |
| 落   | E8 90 BD           | 232 144 189      |
| 的   | E7 9A 84           | 231 154 132      |

这些编码能让计算机区分不同的字符符号，但它们只是“代号”。例如“空”和“落”的 UTF-8 编码放在一起，并不会自动告诉计算机它们组成了“空落落”这种心理感受；“我”“心”“里”“空”“落”“落”“的”的编码连续出现，也不会自动告诉计算机它们组成了“我心里空落落的”这样的表达。

如果词里第二个“落”再次出现，它的字符编码仍然相同。编码能记录符号重复，却不会自动解释这种重叠表达为什么常带有情绪色彩。

也就是说，每个字符在计算机中都有一个单纯用于区分不同字符符号的编码。这个编码没有语义关系，不会表示“空落落”偏向失落，也不会告诉计算机“我心里空落落的”是在表达一种心理状态。

### 3. 大模型通常先把文字切成 Token

大语言模型处理文字时，通常不会直接把整句话当成一个整体，也不一定严格按“一个汉字一个单位”或“一个词语一个单位”来处理。它会先使用分词器，把输入文字切成一个个更小的片段，这些片段常叫做 token，中文翻译成“词元”，它可以理解为是描述一个特定的概念的词语。

例如：

```text
我心里空落落的。
```

在不同分词器中，可能被切成：

```text
我 / 心里 / 空落落 / 的 / 。
```

也可能被切成：

```text
我 / 心 / 里 / 空 / 落落 / 的 / 。
```

英文、数字、标点、空格也可能被切成 token。token 不是固定等于“字”或“词”，而是模型实际处理文本时使用的基本片段。

分词器会把每个 token 转成一个编号，再交给模型继续计算。也就是说，大模型看到的不是人类眼中的完整句子，而是一串 token 编号。后面讲“上下文长度”和“使用成本”时，很多时候也是按 token 数量来计算的。

Token 解释 AI 的"蠢"：算术错/数不清字母，是感知方式决定的

### 4. 词语之间其实有关系

语言中的词不是孤立的。很多词之间有相似、相反、类别、场景等关系：

- 进步、努力、表扬都和“学习变好”有关；
- 考试、目标分数、成绩都和“学习评价”有关；
- 开心、高兴、快乐意义接近；
- 开心和悲伤意义相反。
- 担心、紧张、焦虑经常出现在相似语境中。

AI 要处理语言，就需要把这些关系也变成数字之间可以计算的关系。

### 5. 什么是词向量

词向量就是把一个词变成一串数字。这串数字不只是编号，还尽量包含词语的意义特征，把词语变成数字的这个过程叫做词嵌入（Embedding），嵌入是指把语义嵌入向量空间。

用多个维度描述一个学生：

```text
语文水平、数学水平、体育水平、音乐水平……
```

同样用多个维度描述一个词：

```text
是否和学习有关、情感是积极还是消极、是否表示压力、是否表示社交支持……
```

两个词在很多维度上接近时，它们的词向量也会比较接近。这样，机器就能计算“担心”和“焦虑”比“担心”和“甜品”更相似，也能发现“笑了很久”和“我心里空落落的”在同一段话中同时出现时，可能表示复杂情绪。

**提示：** 可以把一些词投影到二维平面进行聚类，直观展示词语相似性

### 6. 为什么这很重要

大语言模型不能直接把汉字当作人类意义来处理。它要先把文字切分并转化成数字表示，再交给神经网络计算。文字数字化是语言进入 AI 系统的入口。

## 互动强化

### 游戏一：语义运算比拼

**5-10 分钟活动：像做数学一样做“词语关系计算”**

活动目标：让学生直观体验词向量中的一个重要想法：

> 词语之间的关系也可以被表示成方向。  
> 如果两个关系相似，就可以做出类似“北京 - 中国 + 美国 ≈ 华盛顿”的语义推理。

教师先在黑板写出示例：

```text
开心 - 顺利 + 受挫 = ？
```

引导学生发现：如果“开心”和“顺利”经常一起出现，把“顺利”换成“受挫”，更可能得到“难过”“沮丧”或“失落”。这不是精确数学，而是在寻找词语关系。

教师提醒：

> 这不是严格的数学计算，而是在寻找词语之间相似的关系。有些题目可能有多个合理答案，只要关系说得通，就可以接受。

#### 1. 游戏规则

1. 教师逐题出示语义计算题。
2. 学生举手回答，并说明“为什么这样算”。
3. 如果出现多个相近答案，教师可以都写在黑板上，让全班比较哪个更贴近题目关系。
4. 每题控制在 30-60 秒，总时间约 5-10 分钟。

#### 2. 语义计算题

| 题目 | 参考答案 | 关系提示 |
| ---- | -------- | -------- |
| 开心 - 顺利 + 受挫 = ？ | 难过 / 沮丧 / 失落 | 情绪与事件结果 |
| 北京 - 中国 + 美国 = ？ | 华盛顿 / 华盛顿特区 | 首都与国家 |
| 巴黎 - 法国 + 日本 = ？ | 东京 | 首都与国家 |
| 伦敦 - 英国 + 韩国 = ？ | 首尔 | 首都与国家 |
| 人民币 - 中国 + 美国 = ？ | 美元 | 货币与国家 |
| 日元 - 日本 + 欧元区 = ？ | 欧元 | 货币与地区 |
| 国王 - 男性 + 女性 = ？ | 女王 | 性别关系 |
| 王子 - 男性 + 女性 = ？ | 公主 | 性别关系 |
| 爸爸 - 男性 + 女性 = ？ | 妈妈 | 家庭角色 |
| 叔叔 - 男性 + 女性 = ？ | 阿姨 / 姑姑 / 婶婶 | 家庭称谓可能多解 |
| 医生 - 医院 + 学校 = ？ | 老师 / 校医 | 职业与场所，可能多解 |
| 汽车 - 公路 + 铁轨 = ？ | 火车 / 高铁 | 交通工具与行驶场景 |
| 鸟 - 天空 + 水里 = ？ | 鱼 / 鸭子 | 动物与生活环境，可能多解 |
| 钢笔 - 写字 + 画画 = ？ | 画笔 / 铅笔 | 工具与用途 |
| 冰箱 - 冷藏 + 加热 = ？ | 微波炉 / 烤箱 | 家电与功能 |
| 夏天 - 炎热 + 寒冷 = ？ | 冬天 | 季节与典型特征 |

#### 3. 教师追问

每做完几题，教师可以追问：

1. 哪些题只有一个比较明确的答案？
2. 哪些题可以有多个答案？
3. 多个答案都合理时，说明词语之间的关系是不是总能被唯一确定？
4. 如果 AI 只根据大量文本学习这些关系，它会不会也遇到模糊答案？

教师总结：

> 词向量能让机器把词语关系放进数字空间里计算，但语言本身有模糊性。同一道语义运算题可能有多个接近答案，所以 AI 的语言理解也常常是“根据上下文选择最可能的答案”。

### 游戏二：词向量训练模拟

**10-15 分钟活动：让词语在教室里“训练”出位置**

活动目标：让学生用站位变化体验词向量训练中的两个基本动作：

> 正样本：相关词靠近。  
> 负样本：不相关词远离。

#### 1. 词卡设置

只保留 4 类词，每类 3 个词，共 12 张词卡。每张词卡由 1 名学生拿在手中。词卡全部来自或贴近课程案例。

| 类别 | 词卡 |
| ---- | ---- |
| 积极情绪类 | 开心、放松、满足 |
| 消极情绪类 | 焦虑、失落、空落落 |
| 场景类 | 教室、路上、家里 |
| 事件类 | 考试、表扬、合照 |

如果学生人数不足，可以让一名学生负责多张词卡；如果人数较多，其他学生担任“观察员”，记录哪些词逐渐靠近、哪些词逐渐远离。

**注意：** 在发放词卡的时候要打乱随机发放，不要按类别发放，让学生也随机安排站位。

#### 2. 训练句子

教师提前准备下面这些句子，课堂中逐句读出。每句话里加粗的词都是本轮要移动的词卡词；课堂投影时也可以不加粗，让学生自己听出哪些词卡出现了。

1. 听到最近的努力终于有结果，我感到**开心**、**放松**，也有一点**满足**。
2. 想到还有很多任务没完成，我开始**焦虑**，有些**失落**，心里也**空落落**的。
3. 从**教室**出来走在**路上**，回到**家里**后，我还在想今天发生的事。
4. **考试**结束后，老师当众**表扬**了几位同学，大家还拍了一张**合照**。
5. 收到好消息时我很**开心**，可想到明天还有很多任务，又开始**焦虑**。
6. 看到自己的努力有结果，我很**满足**；想到还有差距，又有点**失落**。
7. 和同学聊完后我**放松**了一些，安静下来时心里还是**空落落**的。
8. 早上走进**教室**，放学走在**路上**，晚上回到**家里**，我发现不同地方都会影响自己的心情。
9. 这次**考试**结束后，老师给了我**表扬**，同学们还一起拍了**合照**。
10. 原本我很**焦虑**，听到同学鼓励后慢慢**放松**，心里也多了一点**满足**。
11. 刚开始我有些**失落**，后来想通了又觉得**开心**，但那种**空落落**的感觉还没有完全消失。
12. 我在**家里**翻看那张**合照**，想起那次**考试**，心里一会儿**开心**，一会儿又有点**失落**。

第 1-4 句分别让四个类别内部的词靠近；第 5-7 句故意把积极情绪词和消极情绪词放进同一类情绪语境中，让两组情绪词整体上比和场景类、事件类更靠近；第 8-12 句继续加强场景类、事件类和两类情绪词之间的层次关系。

#### 3. 随机初始化

时间：1-2 分钟

让 12 名学生每人拿一张词卡，随机站在教室中。教师说明：

> 训练开始前，模型还不知道词义，所以词的位置是随机的。

#### 4. 每轮训练：正样本靠近，随机负样本远离

时间：8-10 分钟

教师依次读训练句子。每读一句，完成一轮训练：

1. 先找出句子中出现的词卡词，让它们互相靠近一步。如果一句话里有三个词卡词，可以让三个学生向中间靠拢，也可以让它们两两靠近。
2. 再请没有参与当前句移动的学生或观察员，随机说出 2-3 个没有出现在当前句中的词卡词，作为本轮负样本词。第 5-7 句是情绪桥接句，负样本可以优先从场景类和事件类中随机抽取。
3. 教师把负样本词分别和当前句中的词卡词临时配对。每组成对词互相远离一步。如果当前句只有两个词卡词，抽 2 个负样本即可。

示例：

```text
听到最近的努力终于有结果，我感到开心、放松，也有一点满足。
```

这一轮让“开心”“放松”“满足”互相靠近一步，因为它们都表示积极情绪。

接着，其他学生可以随机说出“教室”“考试”“合照”这 3 个没有出现在当前句中的词。教师把它们临时配成：

```text
开心-教室
放松-考试
满足-合照
```

这三组词互相远离一步，表示模型也在学习“哪些词没有在这个上下文中一起出现”。

再例如：

```text
收到好消息时我很开心，可想到明天还有很多任务，又开始焦虑。
```

这一轮让“开心”和“焦虑”靠近一步。它们一个是积极情绪，一个是消极情绪，但都在表达人的心理感受。随后其他学生可以从场景类和事件类中随机说出“路上”“考试”，教师把它们和当前句中的词组成负样本，例如“开心-路上”“焦虑-考试”，对应词互相远离一步。

经过几轮后，相关词会慢慢聚在一起，不相关词会逐渐被推开。

学生会逐渐观察到：

- 开心、放松、满足逐渐靠近；
- 焦虑、失落、空落落逐渐靠近；
- 教室、路上、家里逐渐靠近；
- 考试、表扬、合照逐渐靠近；
- 因为第 5-7 句的桥接作用，积极情绪类和消极情绪类会形成两个相邻的小团，整体上比和场景类、事件类更靠近。

教师说明：

> 每个句子既提供正样本，也提供负样本。句中一起出现的词会被拉近；没有在当前上下文中一起出现的随机词，会被当作负样本推远。

教师可以提示学生：

> 积极情绪类和消极情绪类都表示情绪类别，语义上比和场景类、事件类更相近。它们互相有区别，所以不会完全混成一团；但整体距离会比和其他类别更近。

#### 5. 观察训练结果

时间：2-3 分钟

让学生观察现在的站位，并回答：

1. 哪些词聚在了一起？
2. 哪些词被拉远了？
3. 现在“开心”最近的词是谁？
4. 现在“焦虑”最近的词是谁？
5. 积极情绪类和消极情绪类，是不是比它们和场景类、事件类更靠近？

预期结果：

- 开心、放松、满足形成积极情绪区；
- 焦虑、失落、空落落形成消极情绪区；
- 教室、路上、家里形成场景区；
- 考试、表扬、合照形成事件区；
- 积极情绪区和消极情绪区彼此相邻，说明它们都属于“情绪”这个更大的语义范围。

#### 6. 教师总结

教师可以用这段话收束：

> 词向量训练不是直接告诉机器“开心是积极情绪”“焦虑是消极情绪”。机器通过大量句子发现：开心、放松、满足经常出现在相似上下文里，焦虑、失落、空落落也经常出现在相似上下文里，所以它们会各自靠近。桥接句又让两类情绪词出现在相近的心理感受语境中，所以积极情绪和消极情绪不会完全混在一起，但会比场景类、事件类更接近。反复训练之后，词向量空间里就会自然形成这种有层次的语义关系。



## 课后提升

**20 分钟小作业：制作 5 个词的迷你词向量**

任选 5 个词，为它们设计 4 个数字维度，例如“是否表示情绪”“情感积极程度”“是否表示场景”“是否表示事件”，每个维度 0-5 分。建议优先使用“开心、放松、焦虑、空落落、教室、考试、合照”等词。

完成后回答：

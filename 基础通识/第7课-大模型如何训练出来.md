# 第7课：大模型如何训练出来？

## 内容引入

**3 分钟互动：如果训练一个作文高手**

教师提问：“如果想把一位同学训练成作文高手，只让他背一篇范文够不够？”学生通常会回答不够。教师继续追问：还需要大量阅读、练习写作、老师批改、学习不同题型。

引导学生把这个过程类比到大模型训练：大模型也不是突然会回答问题，而是经过多阶段训练。

## 知识点讲授

### 1. 大模型“大”在哪里

大模型的“大”主要体现在：

- 参数多；
- 训练数据多；
- 计算量大；
- 能处理的任务多。

可以类比为：普通模型像只读过几本书的学生，大模型像读过海量书籍、网页、代码和资料的学生。但即使读过很多材料，它仍然可能理解错题目或记错事实。

### 2. 预训练

预训练可以理解为：

> 让模型阅读大量文本，反复练习“根据前文预测后文”，从中学习语言规律和世界知识。

例如：

```text
中国的首都是……
```

模型在大量文本中反复看到类似表达，因此学会后面常见答案是“北京”。预训练让模型获得基础语言能力和部分常识。

### 3. 微调

预训练后的模型不一定会很好地听指令。它可能会续写文字，但不一定知道如何按用户要求回答、总结、解释或改写。

微调可以理解为：

> 教模型按照人类希望的方式回答问题。

例如：

- 回答要有条理；
- 不要输出危险内容；
- 不知道时要承认不确定；
- 按用户要求改变风格；
- 面对学习问题时给出清晰步骤。

### 4. 人类反馈

人类反馈是让人评价模型的不同回答，告诉模型哪种回答更好。例如一个回答更准确、更礼貌、更清楚，另一个回答虽然流畅但有事实错误，人类会偏好前者。

模型通过这种反馈继续调整，使输出更符合人类偏好。

### 5. 数据、算力和能源

训练大模型需要大量文本数据、高性能芯片、很长训练时间和高昂成本。它不是单靠一个聪明算法就能完成，而是数据、算法、算力和工程系统共同作用的结果。

## 互动强化

**20 分钟活动：训练一个“班级回答模型”**

### 环境一：每个学生都可以使用自己的电子设备

1. 教师给出一个问题：“怎样安排一次高效的周末复习？”
2. 学生用设备提交一条回答，作为“原始模型输出”。
3. 小组按照标准评分：是否清楚、是否可执行、是否有条理、是否有不合理建议。
4. 每组选择一个回答进行改写，模拟“微调”。
5. 全班投票选出更好的回答，模拟“人类反馈”。
6. 教师串联：大量阅读像预训练，按要求改写像微调，投票偏好像人类反馈。

### 环境二：只有老师可以使用电子设备

1. 教师投影 3 段不同质量的回答。
2. 学生先判断哪段最像“没受过指令训练”的回答，哪段更符合要求。
3. 教师带领全班逐项修改一段回答，使其更清楚、更安全、更有条理。
4. 学生投票选择修改前后哪个更好。
5. 教师总结大模型训练的三阶段类比：预训练、微调、人类反馈。

### 环境三：不能使用电子设备

1. 教师口头给出问题，每组在纸上写一段回答。
2. 各组交换纸条，用“准确、清楚、有条理、符合要求”四项打分。
3. 原组根据反馈修改回答。
4. 教师选取两版回答进行对比，强调反馈如何改善输出。
5. 总结：训练不是一次完成，而是不断输入样本、比较结果、调整表现。

## 课后提升

**20 分钟小作业：画出大模型训练流程图**

请画出从“海量文本”到“可对话大模型”的简化流程图，至少包含：

```text
数据 → 预训练 → 微调 → 人类反馈 → 可使用的大模型
```

每一步用一句话解释作用。有能力的同学可以查阅“大模型训练需要哪些资源”，补充数据、算力、芯片或能源方面的一个事实。

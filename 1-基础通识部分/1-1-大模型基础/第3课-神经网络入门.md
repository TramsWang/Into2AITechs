# 第3课：神经网络入门

## 前一节课内容回顾

**3 分钟回顾：从“数据中找规律”到“更复杂的模型”**

上一节课我们学习了机器学习的基本思想：传统程序依靠人提前写好明确规则，而机器学习是让机器通过大量样本和标签总结规律，再用训练得到的模型预测新样本。几个关键词需要学生重新说出含义：样本是给机器看的例子，标签是正确答案，训练是不断调整判断方式，模型是训练后用于预测的工具。

教师公布课后拓展参考答案：上一节课中的学习情绪分类、猫狗识别都属于有监督学习，因为训练数据里不仅有样本，还有对应的正确标签，例如“这段日记是焦虑”“这张图是猫”。机器通过样本和标签之间的对应关系，学习以后怎样给新样本预测答案。无监督学习则不提前给标准答案，机器面对的是一批没有标签的数据，它要自己寻找相似性、分组或隐藏规律。可以用一句话概括区别：有监督学习像“带答案做例题”，无监督学习像“没有答案，先自己观察这些数据可以怎样分组”。

作业点评时，可表扬目标清楚、样本和标签对应准确的分类任务。例如优秀作业会先说明“要让机器判断垃圾属于哪一类”，再列出纸巾、塑料瓶、果皮、电池等样本及标签，并写出机器可能依据“材质、是否可回收、是否有害”等特征学习规律。这类作业优秀之处在于把“想让机器做什么”和“机器凭什么判断”都写清楚了。

承接上一课的问题：如果任务只是根据一个明确条件判断结果，简单规则也许够用；但学习日记情绪判断、图片识别、语音识别、翻译和写作会同时涉及大量线索。机器怎样综合这么多线索做判断？这节课进入神经网络。

## 内容引入

**3 分钟互动：一句日记为什么不只看关键词？**

教师展示一段匿名学习日记，先不公布标准判断，让学生快速说出这段话的感情基调更接近“开心”“难过”“焦虑”还是“复杂情绪”：

```text
今天考试成绩出来了，我比上次进步了一点，老师也说我最近很努力。放学后朋友们拉着我去吃甜品，还一直讲笑话逗我开心，我也跟着笑了很久。可是回家的路上，我想起自己离目标分数还差很多，又担心爸妈会失望。手机里弹出朋友发来的合照，照片里我和他们笑得很灿烂，但我盯着看了半天，心里还是有点空落落的。
```

学生给出判断后，教师追问依据：有人会关注“进步了一点”“老师表扬”“吃甜品”“讲笑话”“笑了很久”等偏开心的线索，有人会关注“离目标分数还差很多”“担心爸妈会失望”“心里还是有点空落落的”等偏难过的线索，也有人会注意表面行为和真实感受并不完全一致。不同学生会关注不同线索，因此可能给出不同判断。

接着把它放入课程案例：如果校园学习情绪助手收到这段日记，它不能只输出“开心”或“难过”，还要说明依据，并尽量生成合适回应。它至少要处理三个问题：

1. 哪些词语是积极线索，哪些是压力或失落线索？
2. “可是”之后的信息是不是比前面的表面开心更关键？
3. 最终应该判断为单一情绪，还是复杂情绪？

引导学生思考：判断一段话的情感基调时，人类不会只看一个关键词，而是会综合词语、语气、转折和上下文。机器也需要一种能综合复杂线索的模型。

## 知识点讲授

### 1. 为什么需要神经网络

简单问题用简单规则解决，例如分数是否及格、年龄是否达到标准、温度是否超过某个数值。但复杂问题很难靠几条规则解决：

- 判断一段话的情感基调是开心还是难过；
- 翻译一段文字；
- 写一篇结构完整的文章；
- 从语音中识别出文字。

本节课用同一个例子贯穿讲解：让校园学习情绪助手判断一段学习日记的情感基调。

例如：

```text
今天我收到了朋友准备的生日惊喜，大家一起唱歌、吃蛋糕，我觉得特别温暖。
```

大多数人会判断这段话偏“开心”。但如果只写一条简单规则，例如“出现生日就是开心”，就很容易出错：

```text
今天是我的生日，可是没有人记得，我一个人坐了很久。
```

这段话也出现了“生日”，但整体更接近“难过”。同样，主文本中出现了“笑了很久”，但后面又出现“担心爸妈会失望”和“心里空落落”。所以复杂判断不能只看一个词，而要综合很多线索：哪些词出现了、这些词怎样组合、有没有转折、整段话的氛围是什么。神经网络适合处理这类需要综合多种线索的问题。

### 2. 神经网络像什么

神经网络可以类比为一个“班级投票系统”。假设全班一起判断主文本中的同学到底是什么状态，不同同学可以分别关注不同线索：

- 有同学看到“进步了一点”“老师说我很努力”，觉得有开心线索；
- 有同学看到“吃甜品”“讲笑话”“笑了很久”，觉得表面状态偏开心；
- 有同学注意到“可是”，发现后半段可能反转；
- 有同学看到“离目标分数还差很多”“担心爸妈会失望”，觉得有焦虑线索；
- 有同学看到“照片里我和他们笑得很灿烂”“心里还是有点空落落”，觉得表面行为和真实感受不一致。

最后把许多判断综合起来，就会得到更可靠的结果。神经网络中的每个节点也类似：每个节点只做一点简单计算，许多节点连接在一起，就能完成复杂判断。

**引申：** 神经网络不是死记硬背某个关键词，而是在学习数据中更复杂的分布和组合关系。（线性 -> 非线性）

### 3. 输入层、隐藏层、输出层

神经网络分成三类部分。仍然以“判断学习日记情绪”为例：

```text
输入层：接收这段话中的信息
隐藏层：加工和组合这些信息
输出层：给出“开心”“难过”“焦虑”“复杂情绪”等判断
```

可以把流程写成：

```text
一段学习日记 → 神经网络处理 → 输出情绪判断
```

输入层接收文字信息。为了让机器处理文字，通常要先把文字转成数字，例如把词语、词的位置、词与词之间的关系变成模型能理解的表示。

隐藏层负责逐步加工这些信息：

- 第一层可能注意到一些局部线索，例如“进步”“笑了很久”“目标分数”“担心”“空落落”；
- 后面的层会组合更复杂的线索，例如“前面有开心事件，但‘可是’后出现压力和失落”；
- 更深的层会形成整体判断：这段话不是单纯开心，也不是单纯难过，而是更接近复杂情绪。

输出层给出最后结果。它可以不是简单说一个答案，而是给出多个类别分数：

```text
开心：0.22
难过：0.25
焦虑：0.21
复杂情绪：0.32
```

分数更高的一类就是模型的预测结果。

**案例：** 参考CNN隐藏层识别人脸特征的直观案例

### 4. 参数和训练

课堂中可以把参数解释为“每条信息的重要程度”。在判断情感时，有些线索贡献更大，模型就会给它们更高权重。

例如：

```text
今天笑了很久，可是回家路上心里还是有点空落落的。
```

如果模型只把“笑了很久”看得很重要，就可能错误判断为“开心”。训练的作用就是让模型通过大量样本慢慢调整参数：

1. 给模型看很多带标签的例子，例如“这段话是开心”“这段话是难过”“这段话是焦虑”“这段话是复杂情绪”；
2. 模型先根据当前参数做预测；
3. 如果预测错了，就调整参数；
4. 反复多次后，模型会逐渐学到哪些线索更重要、哪些组合更关键。

经过训练后，模型不仅会关注“笑”这类单个词，也会学会重视“可是”“担心爸妈会失望”“心里空落落”这样的上下文。也就是说，训练让神经网络从大量例子中学会更合理的判断方式。

还要提醒学生：参数不是一张“答案表”。神经网络通常不是把每个训练样本原封不动地存起来，而是把大量样本中的规律压缩到参数里。它可能记住某些常见表达或高频知识，但核心仍然是根据输入和参数进行计算。这一点有助于后面理解：大模型看起来像“记得很多知识”，但它不是一本逐条查找的百科全书。

### 5. 神经网络和人脑有什么相似

“神经网络”这个名字借鉴了人脑神经元的启发。课堂中可以先用人脑做类比：人脑里有很多神经元，每个神经元会接收一些信号，再把处理后的信号传给其他神经元。大量神经元连接在一起，人就能综合许多信息做判断。

神经网络也有类似的思路：每个节点接收一些数字信息，进行简单计算，再把结果传给下一层。很多节点连接起来，就能把零散线索组合成整体判断。

在“判断学习日记情绪”的例子里，可以这样类比：

- 人看到“笑了很久”“吃甜品”，会感到这段话有开心的线索；
- 人看到“担心爸妈会失望”“心里空落落”，又会感到这段话有难过的线索；
- 人脑会把这些线索综合起来，形成一个整体印象；
- 神经网络则把这些文字线索转成数字，通过很多节点和参数计算，最后输出不同情绪标签的概率。

但也要提醒学生：神经网络不是人脑的复制品。它不会真正感受到快乐或伤心，它做的是数学计算。这个类比的作用是帮助理解“许多简单节点连接起来，可以共同完成复杂判断”。

## 互动强化

**20 分钟活动：班级情绪判断神经网络**

让学生组成一个“神经网络”，根据学习日记中的线索判断情绪类别。活动目标不是追求心理判断的绝对正确，而是让学生体验：输入特征如何被组合，参数如何影响输出，训练为什么需要根据错误调整。

### 1. 四个输出类别

- 开心
- 难过
- 焦虑
- 复杂情绪

### 2. 输入层：直接线索打分

每组阅读主文本，并记录 5 类输入特征。词汇数量可以按找到的词语或短语个数记录；词汇位置可以把整段文字从开头到结尾标成 0-10，越靠后数字越大。

| 编号 | 输入特征 | 打分说明 |
|------|----------|----------|
| A | 积极词汇数量 | “进步、努力、甜品、笑、开心、灿烂”等积极词汇或短语出现越多，数值越高 |
| B | 积极词汇平均位置 | 所有积极词汇在文本中的平均位置，越靠后数值越高 |
| C | 消极词汇数量 | “差很多、担心、失望、空落落”等消极词汇或短语出现越多，数值越高 |
| D | 消极词汇平均位置 | 所有消极词汇在文本中的平均位置，越靠后数值越高 |
| E | 转折词汇位置 | “可是、但是、然而”等转折词的位置，多个转折词可以取最明显的一个或取平均值 |

主文本可以给出一组参考打分：

```text
A = 6
B = 4
C = 4
D = 8
E = 5
```

### 3. 隐藏层：组合线索

隐藏层不直接输出情绪，而是先组合出更抽象的判断线索。

| 组合特征 | 计算公式 | 含义解释 |
|----------|----------|----------|
| F（积极度） | `w1×A + w2×B` | 积极词汇越多，且积极线索位置越靠后，积极度越高（靠后的内容会被强调） |
| G（消极度） | `w3×C + w4×B` | 消极词汇越多，且消极线索位置越靠后，消极度越高 |
| H（转折强度） | `w5×(E-B)×(D-E)/(\|D-B\| + 1)` | 转折词如果夹在积极线索和消极线索之间，说明前后情绪变化更明显；分母加 1 可以避免除以 0 |

这里在分母外加 1，是为了避免课堂样本中出现 `D = B` 时无法计算，重点仍然是观察参数和位置关系怎样影响结果。

使用课堂初始参数代入计算：

```text
F = 1.0×6 + 0.5×4 = 8
G = 1.0×4 + 0.25×4 = 5
H = 1.0×(5-4)×(8-5)/(|8-4| + 1) = 0.6
```

### 4. 输出层：情绪类别得分

输出层把隐藏层结果再组合成四类情绪分数。

| 类别 | 公式 | 设计逻辑 |
|------|------|----------|
| 开心 | `w6×F - w7×G - w8×H` | 积极度越高越开心，消极度和转折强度会削弱开心判断 |
| 难过 | `w9×G - w10×F - w11×H` | 消极度越高越难过，积极度和转折强度会削弱单纯难过判断 |
| 焦虑 | `w12×F + w13×G - w14×H` | 积极线索和消极线索同时存在时可能形成紧张拉扯，但强转折会削弱单纯焦虑判断 |
| 复杂情绪 | `w15×(F + G) / (\|F - G\| + 1) + w16×H` | 积极度和消极度同时较高且差距不大时，复杂情绪更明显；转折强度会进一步提高复杂情绪得分 |

参数先只定义含义，不急着给数值：

| 参数 | 定义 |
|------|------|
| w1 | 积极词汇数量 A 对积极度 F 的影响 |
| w2 | 积极词汇平均位置 B 对积极度 F 的影响 |
| w3 | 消极词汇数量 C 对消极度 G 的影响 |
| w4 | 积极词汇平均位置 B 对消极度 G 的影响 |
| w5 | 转折位置关系对转折强度 H 的影响 |
| w6、w7、w8 | 开心得分中 F、G、H 的权重 |
| w9、w10、w11 | 难过得分中 G、F、H 的权重 |
| w12、w13、w14 | 焦虑得分中 F、G、H 的权重 |
| w15、w16 | 复杂情绪得分中“积极和消极的平衡程度”与 H 的权重 |

一组比较理想的取值可以是：

```text
w1 = 1.0, w2 = 0.5, w3 = 1.0, w4 = 0.25, w5 = 2.0
w6 = 1.0, w7 = 0.8, w8 = 0.8
w9 = 1.0, w10 = 0.5, w11 = 0.5
w12 = 0.6, w13 = 0.9, w14 = 0.8
w15 = 1.5, w16 = 3.0
```

这组值只是教师参考，表示如果目标是让主文本更容易被判断为“复杂情绪”，模型需要重视积极和消极并存，也要重视转折词夹在前后情绪线索之间。

实际授课时，不要一开始就给理想取值。可以先给一组不太理想的初始取值，让学生看到模型会出现偏差，再在互动过程中逐步迭代：

```text
w1 = 1.0, w2 = 0.5, w3 = 1.0, w4 = 0.25, w5 = 1.0
w6 = 1.0, w7 = 0.4, w8 = 0.3
w9 = 1.0, w10 = 0.5, w11 = 0.4
w12 = 0.7, w13 = 1.0, w14 = 0.2
w15 = 0.8, w16 = 0.8
```

用这组初始参数计算主文本：

```text
开心 = 1.0×8 - 0.4×5 - 0.3×0.6 = 5.82
难过 = 1.0×5 - 0.5×8 - 0.4×0.6 = 0.76
焦虑 = 0.7×8 + 1.0×5 - 0.2×0.6 = 10.48
复杂情绪 = 0.8×(8+5)/(|8-5|+1) + 0.8×0.6 = 3.08
```

初始模型会把它判为“焦虑”。教师追问：这个结果是否合理？学生可能认为“焦虑”有道理，但“复杂情绪”也很接近，因为这段日记同时包含被肯定、社交陪伴、成绩压力和内心失落。

### 5. 迭代修正参数

教师给出一条简单调参规则：

```text
如果正确类别分数偏低：提高正确类别对关键组合特征的权重。
如果错误类别分数偏高：降低错误类别对过强线索的依赖。
每轮只改少数几个参数，观察是否修正了错误。
```

如果全班讨论后希望把主文本标为“复杂情绪”，可以向理想取值靠近。第一步先提高转折强度和复杂情绪权重：

```text
调整前：w5 = 1.0, w15 = 0.8, w16 = 0.8
调整后：w5 = 2.0, w15 = 1.5, w16 = 3.0
```

重新计算：

```text
H = 2.0×(5-4)×(8-5)/(|8-4| + 1) = 1.2
复杂情绪 = 1.5×(8+5)/(|8-5|+1) + 3.0×1.2
        = 4.875 + 3.6
        = 8.475
```

这次复杂情绪分数明显升高。教师可以继续追问：焦虑得分是否仍然过高？因为“可是”不一定只代表焦虑，也可能代表情绪混合。

第二步再调整焦虑相关参数，让它更接近理想取值：

```text
调整前：w12 = 0.7, w13 = 1.0, w14 = 0.2
调整后：w12 = 0.6, w13 = 0.9, w14 = 0.8
```

重新计算：

```text
焦虑 = 0.6×8 + 0.9×5 - 0.8×1.2
     = 4.8 + 4.5 - 0.96
     = 8.34
```

此时复杂情绪分数超过焦虑，模型会把主文本判为“复杂情绪”。教师可以说明：真实模型不会只靠一条样本调参，而会用大量标注样本反复训练。课堂里的简化公式只是帮助学生看见“参数影响判断结果”。

### 6. 用新样本复测

为了避免只为主文本改参数，教师再给出两个测试样本。

```text
样本1：这次比赛我拿了一等奖，老师和同学都替我高兴。我回家后还把奖状贴在书桌前，觉得这段时间的努力很值得。
参考标签：开心
```

```text
样本2：明天就要考试了，我明明复习了很久，却总觉得还有很多不会。晚上躺下后一直在想万一考砸怎么办。
参考标签：焦虑
```

学生用同一套输入层、隐藏层和输出层计算，检查参数调整后是否仍能正确判断其他样本。最后总结：训练不是一次性写出完美公式，而是根据大量错误样本不断调整参数，使正确类别得分更高、错误类别得分更低。
